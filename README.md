# Фитнес-трекер | Проект учебной практики УП.02

## Часть 1. Описание дипломного проекта

### Тема проекта
Веб-приложение «Фитнес-трекер: график тренировок и справочник упражнений»

---

### Актуальность темы
В современном мире многие люди начинают заниматься спортом дома или в зале, но сталкиваются с проблемами: отсутствие системы, непонимание, какие упражнения для каких мышц работают, и как правильно прогрессировать. Существующие приложения перегружены платной подпиской или сложной аналитикой. Разработка простого, наглядного трекера тренировок с графиком и описанием упражнений позволит пользователю структурировать занятия и отслеживать прогресс.

### Целевая аудитория
- **Новички в фитнесе** — которым нужен гайд по упражнениям
- **Люди, тренирующиеся дома** — без тренера, самостоятельно
- **Спортсмены-любители** — для ведения дневника тренировок

### Предполагаемые функциональные возможности

| Категория | Функция |
|---|---|
| **График тренировок** | Создание и редактирование недельного графика |
| **Справочник упражнений** | Каталог с описанием: какие мышцы работают, техника выполнения |
| **Фильтрация упражнений** | По группам мышц (грудь, ноги, спина, плечи, пресс) |
| **Дневник тренировок** | Запись выполненных упражнений (подходы, повторения, вес) |
| **Статистика прогресса** | Отображение динамики при ≥3 записях по упражнению |
| **Локальное сохранение** | Все данные хранятся в браузере (localStorage) |

### Используемые технологии

| Технология | Назначение |
|---|---|
| HTML5 | Семантическая структура |
| CSS3 (Flexbox/Grid) | Адаптивная вёрстка |
| JavaScript (ES6) | Логика, фильтры, localStorage |
| LocalStorage | Хранение данных пользователя |
| Draw.io | Use-case диаграмма |

### Ожидаемый результат
Рабочий прототип веб-приложения, который позволяет:
1. Просматривать и редактировать недельный график тренировок
2. Изучать упражнения с описанием техники и групп мышц
3. Записывать выполненные подходы в дневник
4. Видеть статистику прогресса по упражнениям
5. Сохранять все данные при перезагрузке страницы

## Часть 2. Use-case диаграмма

![Use-case диаграмма](./docs/use-case.png)

### Акторы
- **Пользователь** — единственный актор системы

### Варианты использования (Use cases)

| Use case | Тип | Примечание |
|---|---|---|
| Просмотреть список упражнений | базовый | — |
| Отфильтровать упражнения по мышцам | базовый | — |
| Посмотреть описание упражнения | базовый | клик по карточке |
| Создать/редактировать график | базовый | выбор дня + новое название |
| Добавить запись в дневник | базовый | форма с валидацией |
| Удалить запись из дневника | базовый | кнопка удаления в таблице |
| **Сохранить все данные** | **include** | автоматически после любого изменения |
| **Показать статистику упражнения** | **extend** | если записей ≥3 по одному упражнению |
